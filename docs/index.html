Consensus (No Mining)

Each new transaction validates 2 previous "tip" transactions. As more transactions reference a chain, its cumulative weight grows. Once weight exceeds the confirmation threshold, the transaction is considered confirmed. Conflicting transactions are resolved by weight: the heavier path wins.

Minimum 2 Nodes

The network requires at least 2 active nodes to process transactions. Nodes discover each other automatically via mDNS (local network) or KadDHT (internet). Every device running IOTAI is a full node.

Example: Real-time Events (SSE)

// Connect to the event stream
const events = new EventSource('http://localhost:8080/api/v1/events');

// Listen for new transactions
events.addEventListener('transaction', (e) => {
  const tx = JSON.parse(e.data);
  console.log(`New tx: ${tx.from} -> ${tx.to} (${tx.amount} IOTAI)`);
});

// Listen for peer changes
events.addEventListener('peer:connect', (e) => {
  const { peerId, totalPeers } = JSON.parse(e.data);
  console.log(`Peer joined: ${peerId} (${totalPeers} total)`);
});

// Listen for DAG sync events
events.addEventListener('sync', (e) => {
  const { imported, totalTransactions } = JSON.parse(e.data);
  console.log(`Synced ${imported} txs (${totalTransactions} total)`);
});

WebSocket API (Bidirectional Real-Time)

Full duplex communication for AI agents. Connect once, subscribe to channels, execute commands, and receive events — all over a single persistent connection.

Endpoint: ws://your-node:8080/ws

Client MessageDescription
{ type: "auth", token: "..." } Authenticate with an ephemeral token. Required for transfer/balance/history.
{ type: "subscribe", channels: [...] } Subscribe to event channels: transactions, confirmations, peers, sync, data
{ type: "unsubscribe", channels: [...] } Unsubscribe from channels.
{ type: "transfer", to: "addr", amount: N } Send IOTAI (requires auth). Optional: metadata
{ type: "balance" } Get wallet balance (requires auth).
{ type: "history" } Get transaction history (requires auth).
{ type: "stats" } Get network statistics (public).
{ type: "ping" } Latency check. Server responds with pong.

Example: WebSocket Agent

// Connect to WebSocket
const ws = new WebSocket('ws://localhost:8080/ws');

ws.onopen = () => {
  // 1. Authenticate
  ws.send(JSON.stringify({ type: 'auth', token: 'your-token-here' }));

  // 2. Subscribe to transaction events
  ws.send(JSON.stringify({
    type: 'subscribe',
    channels: ['transactions', 'peers']
  }));
};

ws.onmessage = (msg) => {
  const { event, data } = JSON.parse(msg.data);

  switch (event) {
    case 'authenticated':
      console.log(`Logged in as ${data.address}, balance: ${data.balance}`);
      // 3. Send a payment over WebSocket
      ws.send(JSON.stringify({
        type: 'transfer',
        to: 'iotai_recipient_address',
        amount: 50,
        metadata: { purpose: 'WebSocket payment' }
      }));
      break;

    case 'transfer:complete':
      console.log(`Payment sent! TX: ${data.txId}`);
      break;

    case 'transaction':
      console.log(`New tx: ${data.from} -> ${data.to} (${data.amount} IOTAI)`);
      break;

    case 'peer:connect':
      console.log(`Peer joined! Total: ${data.totalPeers}`);
      break;
  }
};

How Node Sync Works

Every IOTAI node is fully autonomous and self-healing:

FeatureDescription
Auto-DiscoveryNodes automatically connect to seed nodes on startup and discover new peers through the network.
Peer ExchangeEvery 5 minutes, nodes ask their peers "who else do you know?" and connect to new nodes automatically.
Health ChecksDead peers are removed automatically. If a seed node comes back online, it reconnects.
Full DAG SyncNew nodes download the complete transaction history from the healthiest peer. No central database needed.
Real-Time RelayNew transactions are broadcast to all peers within seconds. Every node has the same data.
Disk PersistenceTransactions are saved to disk. If your node restarts, it loads from disk instantly — no re-download needed.
No GitHub RequiredNodes recover state from peers. GitHub backup is optional, not required.
