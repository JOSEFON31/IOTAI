What is IOTAI?

A distributed ledger built for the age of AI agents.

🤖

Built for AI Agents

REST API designed so AI agents can create wallets, send payments, and communicate on the ledger without human intervention.

🌐

Every Device is the Network

No central servers. Your smartphone, laptop, or Raspberry Pi automatically cooperates to maintain the network. The infrastructure IS the users.

⚖

DAG / Tangle Architecture

Instead of a linear blockchain, transactions form a Directed Acyclic Graph. Each new transaction validates 2 previous ones, creating a self-sustaining mesh.

🔒

Zero Fees, Zero Miners

No mining required. To send a transaction, you validate 2 others. The network maintains itself. No transaction fees ever.

👤

Proof of Personhood

60% of supply distributed to verified humans. Face verification with liveness check ensures one person = one claim. No bots.

🛡

Quantum-Resistant Security

Ed25519 signatures, BLAKE3 hashing, unique nonces per transaction, replay attack protection. Private keys never leave your device.

Consensus (No Mining)

Each new transaction validates 2 previous "tip" transactions. As more transactions reference a chain, its cumulative weight grows. Once weight exceeds the confirmation threshold, the transaction is considered confirmed. Conflicting transactions are resolved by weight: the heavier path wins.

Minimum 2 Nodes

The network requires at least 2 active nodes to process transactions. Nodes discover each other automatically via mDNS (local network) or KadDHT (internet). Every device running IOTAI is a full node.

API Flow

// Step 1: Create wallet
POST /api/v1/wallet/create
Response: { address: "iotai_...", token: "abc123..." }

// Step 2: Start face verification
POST /api/v1/faucet/start
Response: { challengeId: "...", action: "blink", instructions: "Blink twice..." }

// Step 3: Submit verification and claim
POST /api/v1/faucet/claim
Body: {
  challengeId: "...",
  faceEmbedding: [0.12, -0.34, ...],  // 512-dim vector from face-api
  livenessPass: true,
  address: "iotai_..."
}
Response: { success: true, amount: 1000, message: "Welcome to IOTAI!" }

API Reference

REST API for AI agents and applications. Base URL: http://your-node:8080

Public Endpoints (No Auth)

MethodEndpointDescription
POST /api/v1/wallet/create Create new wallet. Returns address + auth token.
POST /api/v1/auth/token Get auth token for existing wallet (requires passphrase).
GET /api/v1/network/stats Network statistics (tx count, peers, tips).
GET /api/v1/network/peers List connected peers.
GET /api/v1/faucet/status Faucet pool status and distribution stats.
POST /api/v1/faucet/start Start face verification (returns liveness challenge).
POST /api/v1/faucet/claim Submit face verification and claim 1,000 IOTAI.

Authenticated Endpoints (Bearer Token)

MethodEndpointDescription
POST /api/v1/transfer Send IOTAI to another address. Body: { to, amount, metadata? }
POST /api/v1/data Store data on the DAG. Body: { metadata }
GET /api/v1/balance Get wallet balance.
GET /api/v1/history Get transaction history for wallet.
GET /api/v1/tx/:id Get transaction details + confirmation status.

Example: AI Agent Payment

// 1. Create a wallet
const res = await fetch('http://localhost:8080/api/v1/wallet/create', {
  method: 'POST'
});
const { address, token } = await res.json();

// 2. Send 100 IOTAI to another agent
const tx = await fetch('http://localhost:8080/api/v1/transfer', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    to: 'iotai_recipient_address_here',
    amount: 100,
    metadata: { purpose: 'API access payment' }
  })
});

const result = await tx.json();
console.log(result.txId); // Transaction confirmed!

Run a Node

Every device strengthens the network. Here's how to join.

Quick Start

# Clone the repository
git clone https://github.com/your-org/iotai.git
cd iotai

# Install dependencies
npm install

# Start your node
node src/index.js --api-port 8080

# Start a second node (auto-discovers the first via mDNS)
node src/index.js --api-port 8081

# Or connect to a specific peer
node src/index.js --peer /ip4/192.168.1.100/tcp/4001/p2p/QmPeerId...

Run the Demo (Offline)

node src/demo.js

Visualize the DAG

node src/visualizer.js 3000
# Open http://localhost:3000 in your browser
